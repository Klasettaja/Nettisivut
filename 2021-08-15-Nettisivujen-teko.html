Nettisivujen luonti

Julkaistu 19.08.2021

Tervetuloa blogiini! Tämä on ensimmäinen blogipostaukseni ja aloitan sen kertomalla, kuinka olen luonut nämä nettisivut hyödyntämällä Github-sivuja, Visual Studio Codea ja Githubia. Tämän blogipostauksen ohjeiden avulla myös sinä opit tekemään kustannustehokkaat staattiset nettisivut. Käytännössä Github Pages -palvelu julkaisee sivusi username.github.io sivustolle, josta sisältö ohjataan varaamallesi verkkosivulle. Github Pages- ja DNS-ohjauksien kanssa kannattaa olla kärsivällinen, sillä ensimmäisten tuntien aikana ohjausmuutoksista sivuston toiminta on epävakaata. Kannattaa siis odottaa ainakin vuorokausi ennen kuin epäilee, että on tehnyt jotain väärin. Mainittakoon kuitenkin vielä hyvänä puolena, että Github Pages -tarjoaa sivustollesi SSL-suojauksen (https), jonka avulla sivusto näyttää turvallisena vierailijoille.

Toimenpiteet

Aivan ensiksi sinun tulee varata verkkotunnus käyttöösi joltain palveluntarjoajalta, kuten esimerkiksi domainhotellista sekä hankkia mahdolliset lisäpalvelut, joilla voit muuttaa DNS-nimipalveluasetuksia. Staattisten nettisivujen ylläpito tapahtuu Github Pages -palvelun avulla, jonka IP-osoitteet sinun tulee lisätä DNS-asetuksiisi.

Nettisivujen kehittämistä varten sinun on hyvä ladata myös ohjelmistokehityspaketin sisältävä tekstieditori, kuten Visual Studio Code. Visual Studio Code (VSC) on Microsoftin tarjoama ilmainen Git-versionhallintaa tukeva ohjelmointityökalu. Mikäli Git ei tule VSC-latauspaketin, tulee Git ladata erikseen Gitin nettisivuilta. Kun olet ladannut ohjelmistot, niin asenna VSC-sovellukseen vielä lisäosina (extensions) Git Extension Pack, gitignore sekä Live Server. Git-lisäosat auttavat jatkossa versiohallinan kanssa ja Live Serverin avulla nettisivujen kehitys on puolestaan nopeampaa.

Ohjelmistojen latautuessa voit perustaa Github-tilin osoitteessa https://github.com/. Github-tilille voit täydentää haluamasi osoitetiedot, mutta oleellisinta on perustaa uusi repository nettisivujen kehittämistä varten. Nettisivuja kehitettäessä nettisivujen lähdekoodit sekä niihin liittyvät aineistot julkaistaan repositoryyn, jossa voidaan hallinnoida sivujen versiohallintaa. Lisäksi staattisten nettisivujen ylläpidossa hyödynnettävä Github Pages -palvelu edellyttää, että sivuston lähdekoodit on julkaistu Githubiin, jotta nettisivut toimivat.

Kun olet tehnyt kaikki edellä mainitut perustoimet, niin voit julkaista ensimmäisen versiosi index.html-tiedostostasi luomaasi Github-repositoryyn. Sinun tulee ensin avata uusi projektikansio Visual Studio Codessa ja tämän jälkeen vielä määritellä, että projektikansio kuuluu Git-versionhallinnan piiriin. Lopuksi voit suorittaa push-komennon eli työntää koodin Github-repositoryyn ja silmäillä tuloksia juuri luomillasi nettisivuilla.

Vinkkejä

  • HTML-merkintäkielen opiskelu onnistuu helpoiten sivustolla https://www.w3schools.com/.
  • Nettisivujen luonti kannattaa aloittaa luomalla sivusto jo jotakin valmista ilmaista sivupohjaa hyödyntämällä, mikäli haluaa saada nettisivut pystyyn nopeasti.
  • Nettisivujen kävijämäärien seuranta onnistuu kätevästi Google Analytics työkalulla, kun sivustolle on ensin tiputettu Google-seurantatunnus.

Ylläpitokustannukset

Tämän sivuston ylläpitokustannukset muodostuvat domainhotellin kautta varatusta verkkotunnuksesta sekä DNS-nimipalvelut sisältävästä yhden tähden webhotellista. Sivuston ylläpitokustannukset ovat nämä elementit huomioiden noin 15 € vuodessa.
